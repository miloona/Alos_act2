# Alos_act2


## Code status used

| Code |        status         | Description                                                                                        |
|:----:|:---------------------:|:---------------------------------------------------------------------------------------------------|
| 200  |          OK           | Indicate success                                                                                   |
| 400  |      Bad Request      | Indicate something is not right with the request, for example, missing information or invalid data |
| 404  |       Not Found       | Resource not found                                                                                 |
